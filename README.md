## Getting Started

First, install dependencies:

```bash
npm install
# or
yarn install
```
Run compile command:
```bash
npx hardhat compile
# or
yarn hardhat compile
```

Run deploy scripts:
```bash
npx hardhat deploy
# or
yarn hardhat deploy
```


Run tests:
```bash
npx hardhat test
# or
yarn hardhat test
```

Run persistent local blockchain:
```bash
npx hardhat node
# or
yarn hardhat node
```
